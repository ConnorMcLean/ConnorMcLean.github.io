POS-API

POS-style RESTful API, written in Java with a PostgreSQL database using the Spring and Hibernate frameworks. Tracks customers, orders and current products, with capability to add more, will eventually fill and return PDF templates relating to said data through the PDFbox library. Authentication and the front-end componenets are currently under progress.

MEAN Stack Applicaiton

Basic MEAN stack application created using Angular 6 and a RESTful API with form submission and mongoose data modelling. Completed by following an online guide.

Auto Order Comparison

Basic python scripting to compare two csv files via a master reference sheet and list all discrepencies found between files for later review. Compared both expected cost and amount of parts between an order file and a receipt file. Completed using the Pandas dataframe library. Files were too inconsistent from suppliers to fully complete the project.

UnDelay

A MEAN stack application created to track live delays in the public transport network. This was done via the PTV API and Live data submission. Created during Web Information Techonologies subject in 2017.
